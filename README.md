# Rate Limiter Lab

A Java Spring Boot learning project to design and implement API rate limiting algorithms.

## Goals

- Understand fixed window, sliding window, token bucket, and leaky bucket algorithms
- Build a Spring Boot API with rate limiting
- Add Redis-backed distributed rate limiting
- Add observability using Prometheus and Grafana
- Add structured logs and load testing

## Tech Stack

- Java 21
- Spring Boot
- Maven
- Redis
- Docker
- Prometheus
- Grafana
- k6
